=========== HƯỚNG DẪN DEMO (CHẠY TRÊN MÁY KHÁC) ===========

Bước 1: Copy thư mục này vào máy tính Demo (Tốt nhất là để ngoài Desktop).

Bước 2: Cài Docker Desktop (Nếu máy đó chưa có).
        - Link tải: https://www.docker.com/products/docker-desktop/
        - Cài xong nhớ bật Docker lên.

Bước 3: CLICK ĐÚP vào file "start-app.bat"
        - Chờ khoảng 5-10 phút để nó tự cài đặt (Lần đầu thôi, lần sau 10 giây).
        - Khi nào thấy dòng chữ "System is Starting!" màu xanh là xong.

Bước 4: Mở trình duyệt vào:
        - Web Khách: http://localhost:3000
        - Web Admin: http://localhost:3000/admin (hoặc link anh quy định)
        - API Docs : http://localhost:8080/swagger-ui/index.html

LƯU Ý: KHÔNG CẦN cài Java, Node.js hay SQL gì cả. Chỉ cần Docker là đủ.
